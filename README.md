# Dance Star

Phoenix's home for **Bachata, Salsa and Latin Fusion** — a self-contained, single-file
marketing site with a bold red palette and a warm, community-driven brand voice.

## Preview it instantly

Open `index.html` in any browser. The page is fully self-contained (fonts, smooth-scroll,
map embeds and the live Friday-social countdown load over the network) — no build step.

## Publish a live link (GitHub Pages)

1. Create a new **public** repository (e.g. `dance-star`).
2. Upload these files (including the `.github/` folder) to the `main` branch.
3. The included workflow runs automatically, enables Pages, and publishes to:
   `https://<your-username>.github.io/dance-star/`

No settings toggle is required — `actions/configure-pages` enables Pages on the first run.
(GitHub Pages is free for public repos; private repos need a paid plan.)

### Even simpler, no workflow

Create the repo with just `index.html`, then go to **Settings → Pages → Deploy from a
branch → `main` / `root`**. Same result.

## Brand

- **Voice:** energetic, welcoming, beginner-friendly. "Dress to impress. Dance to connect."
- **Palette:** deep wine/near-black, vivid Latin red (`#E12330`), warm gold, cream.
- **Type:** Playfair Display (headings), Sora (UI), Inter (body), Dancing Script (accent).
